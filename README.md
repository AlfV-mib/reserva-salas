# Reserva de Salas

Aplicación web para la gestión de reservas de las salas de formación y reuniones.

## Acceso

👉 [Abrir la aplicación](https://alfv-mib.github.io/reserva-salas/)

## Uso

Cada usuario puede registrarse con su correo electrónico y contraseña.
Una vez dentro, puede ver el calendario de reservas, crear nuevas reservas
y gestionar las suyas propias.

## Salas disponibles

- Sala de Formación
- Sala de Reuniones

## Horario

Lunes a viernes, 07:00 – 18:00.
